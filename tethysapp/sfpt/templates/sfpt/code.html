This application is heavily inspired by the Streamflow Prediction Tools built by Dr Alan Snow and the improvements made by Dr Michael Suffront. This app is unique because it:

  1. Consumes the SFPT REST API hosted by Microsoft Azure rather than trying to manage local data.
  2. Provides a Global Streamflow Animation map started by Dr Suffront, improved by Hart Henrichson, and adapted for Tethys by Riley Hales.
  3. Gives citations, documentation for the entire streamflow prediction workflow, and FAQ and help sources in-app.
